# Badusb-Badble
BadUSB-BadBLE is a cross-platform penetration testing framework that leverages HID emulation to execute payloads. By combining traditional BadUSB capabilities with modern BLE exploitation, this project allows for seamless, remote, and stealthy command injection on target systems.
